# 🚁 Neon Chopper Command

Um jogo arcade moderno inspirado no clássico **Chopper Command**, desenvolvido em um único arquivo HTML utilizando apenas:

- HTML5 Canvas
- CSS3
- JavaScript ES6+

O projeto foi criado para rodar diretamente no navegador, sem necessidade de backend ou instalação.

---

# 🎮 Gameplay

Controle um helicóptero militar futurista e sobreviva ao maior tempo possível enfrentando:

- Helicópteros inimigos
- Aviões
- Tanques
- Torres antiaéreas
- Mísseis teleguiados

O jogo possui:

- Física arcade fluida
- Efeitos neon modernos
- Sistema de partículas
- Explosões
- Power-ups
- Progressão infinita de dificuldade
- Compatibilidade mobile e desktop
